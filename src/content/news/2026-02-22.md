## [What Is a Database Transaction?](https://stackoverflow.com/questions/974596/what-is-a-database-transaction)
* A database transaction is a group of operations that either all execute successfully (commit) or, if one fails, the system reverts to its state before the transaction began (rollback) [https://stackoverflow.com/questions/974596/what-is-a-database-transaction]
* Transactions ensure data integrity by treating a series of database operations as a single, indivisible unit, preventing inconsistent states that could arise from errors during the process [https://cybernerdie.medium.com/database-transactions-explained-a-deep-dive-into-reliability-17ab4e17117a]
* A database transaction consists of one or more independent units of work that read and/or write information to a database [https://en.wikipedia.org/wiki/Database_transaction]
