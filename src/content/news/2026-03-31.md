## [Axios compromised on NPM – Malicious versions drop remote access trojan](https://www.itnews.com.au/news/supply-chain-attack-hits-300-million-download-axios-npm-package-624699)
* Axios, a widely used HTTP client library with over 300 million weekly downloads, was compromised on the Node Package Manager (NPM) registry [https://www.itnews.com.au/news/supply-chain-attack-hits-300-million-download-axios-npm-package-624699]
* Malicious versions of Axios, specifically 1.14.1 and 0.30.4, included a malicious dependency (plain-crypto-js@4.2.1) and were published outside the project's usual release flow [https://www.reddit.com/r/msp/comments/1s8e1af/new_axios_1141_and_0304_on_npm_are_likely]

## [Open source CAD in the browser (Solvespace)](https://solvespace.com/forum.pl?action=viewthread&parent=6196&tt=1760062167)
